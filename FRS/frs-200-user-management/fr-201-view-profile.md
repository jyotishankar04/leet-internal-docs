# FR-201 — View Profile

#### Actors

* Guest
* Registered User

#### Includes

* Name
* Username
* Avatar
* Bio
* Reputation
* Created Problems
* Solved Problems
* Public Statistics
* Badges

#### Acceptance Criteria

* Private information is never exposed.
* Suspended users are handled appropriately.
* Public profile loads successfully.
