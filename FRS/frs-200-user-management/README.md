# FRS-200 — User Management

**Purpose:** Manage user identity beyond authentication, including profile, reputation, preferences, activity, and personal data.

#### Functional Areas

```
User Management
│
├── Profile
├── Preferences
├── Reputation
├── Statistics
├── Activity
├── Bookmarks
├── Saved Drafts
├── Following (Future)
├── Integrations
└── Privacy
```
