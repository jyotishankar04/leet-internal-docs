# FR-301 — Create Problem

#### Actors

* Registered User

#### Preconditions

* User is authenticated.

#### Main Flow

1. User selects "Create Problem."
2. System creates an empty draft.
3. User is redirected to the editor.

#### Postconditions

* Draft exists.
* Creator becomes owner.
